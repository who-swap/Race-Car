# Race Car 🚗

A simple race car project built with **C** and **raylib** as part of my journey into computer graphics and game development.

## About

This repository contains my experiments while learning:

* Graphics programming with **raylib**
* Game development fundamentals in **C**
* Keyboard input handling
* 2D rendering
* Basic movement and rotation
* Frame-based game loops

The goal of this project is not to create a complete game, but to understand how graphics programming works from the ground up.

## Technologies Used

* C
* raylib
* GCC

## Features

* Car rendering
* Keyboard controls
* Rotation and movement
* Continuous rendering using the game loop

## Learning Goals

Through this project I am learning:

* How game loops work
* Drawing shapes and textures
* Handling user input
* Working with vectors and angles
* Organizing C code for game projects
* Building interactive applications using raylib

## Build

Make sure you have **raylib** installed.

```bash
gcc main.c -o race-car -lraylib -lm -ldl -lpthread -lGL -lrt -lX11
```

Run:

```bash
./race-car
```
